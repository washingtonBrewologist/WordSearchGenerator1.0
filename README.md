# WordSearchGenerator
This program will allow a user to generate a word search by either entering the words manually, or by having the words read in from a file. Words read in from a text file should be in the following format:

Ex: words.txt

Cruise
Travolta
Cage
Dinero
Sosa
McGwire
Bonds
Maris
McCovey
Robinson
Hanks
Sanders


Once a word search has been generated, a solution to the puzzle is availale by selecting 's' from the main menu. Likewise, if a puzzle is saved to a file, a solution will be icluded in the file.
